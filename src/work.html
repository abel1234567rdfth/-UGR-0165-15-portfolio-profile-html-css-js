BookTrack

ReadNice is a web-based book management application that enables users to catalog their reading collection, take detailed notes, and track progress. Overview: BookTrack is a lightweight web-based application designed for book lovers to manage their reading list and track progress. The platform enables users to add and organize books in three simple categories: "Want to Read," "Currently Reading," and "Finished." Users can also leave brief personal notes for each book to capture thoughts or memorable quotes.

TaskBuddy API

TaskBuddy API is a straightforward backend system for managing to-do tasks. It allows users to create, organize, and track tasks efficiently through a simple RESTful API. Key Features: - User Authentication: Secure login with JWT. - CRUD Operations: Create, read, update, and delete tasks. - Task Status Updates: Mark tasks as "Pending," "In Progress," or "Completed."

BudgetEase

BudgetEase is a simple and intuitive frontend application designed to help users track their income and expenses with a responsive interface. Features: - Add Transactions: Input income and expenses by category. - Transaction List: View a categorized list of transactions. - Financial Summary: See income, expenses, and remaining balance.
